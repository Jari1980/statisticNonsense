# StatisticNonsense
<p>This is a Rest Api done in Java Spring, providing data to statistics in front end.</p>
<p>Data is provided from two sources, Csv file and MySqlDB. In order to run this you need to set your local credentials in application.properties for MySql</p>
<p>Dependencies used in this project are:</p>
<ul>spring-boot-starter-web</ul>
<ul>spring-boot-starter-data-jpa</ul>
<ul>spring-boot-devtools</ul>
<ul>mysql-connector-j</ul>
<ul>lombok</ul>
<ul>spring-boot-starter-data-jpa-test</ul>
